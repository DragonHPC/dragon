AI
++

Dragon provides first-class integration with popular AI/ML frameworks and workflows. These examples demonstrate
how to use Dragon's capabilities for distributed training, scalable data loading, LLM inference, and multi-agent
orchestration on HPC clusters.

* **AI-in-the-loop**: Tightly couple simulation and ML inference in an HPC workflow.
* **AI Agents**: Build and run LLM-powered multi-agent pipelines with tools, memory, and tracing.
* **Distributed Data Loading**: Use Dragon-backed PyTorch datasets for scalable, high-performance training.
* **DNA/RNA DataLoader**: Example of domain-specific distributed data loading.
